# Git 0 → Hero — คู่มือฉบับเข้าใจถึงแก่น

คู่มือสอน Git ภาษาไทยแบบ **"0 to Hero"** ในไฟล์ HTML เดียวจบ (self-contained) — เน้น *แบบจำลองความคิด* และ *กลไกการทำงานภายใน* มากกว่าการท่องคำสั่ง

🔗 **อ่านออนไลน์:** https://tayakorn221.github.io/git-handbook/

## เนื้อหา
- **14 บท เรียง 5 ระดับ** ตั้งแต่ติดตั้ง Git จนถึงกลไกภายใน (4 objects, content-addressable, ทำไม commit ถึงแก้ไม่ได้)
- **ไดอะแกรม inline SVG** ทุกบทแนวคิด: สามพื้นที่ของ Git, commit graph, fast-forward vs merge, merge vs rebase, fetch vs pull, โครงสร้าง object
- Sidebar สารบัญ + scrollspy, แถบ reading progress, รองรับมือถือ และพิมพ์ (`@media print`) ได้

## ไฟล์
| ไฟล์ | คืออะไร |
|------|---------|
| `index.html` | ตัวคู่มือ — เปิดด้วยเบราว์เซอร์ได้เลย ไม่ต้องติดตั้งอะไร |
| `PROMPT.md` | สเปกที่ใช้สร้างคู่มือ (เผื่อดัดแปลงไปทำหัวข้ออื่น เช่น Docker / SQL / Linux) |
